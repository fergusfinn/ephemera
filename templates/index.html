Somnial

A simple time-series metrics collection and visualization service. Submit metrics via API and view interactive charts.

Submit Metrics

POST /{namespace}/{id}?value={number}

Submit a metric value with a timestamp.

Example

curl -X POST "https://charts.somnial.co/myapp/cpu_usage?value=45.2"

View Charts

GET /{namespace}/{id}

View an interactive chart of collected metrics over time.

Example

https://charts.somnial.co/myapp/cpu_usage

Browse Namespace

GET /{namespace}

View all charts in a namespace.

Example

https://charts.somnial.co/myapp
