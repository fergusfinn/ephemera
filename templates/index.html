Somnial

A simple time-series metrics collection and visualization service. Submit metrics via API and view interactive charts.

Log your first metric

curl -X POST "https://charts.somnial.co/app/cpu?value=45"

Submit Metrics

POST /{ns}/{chart}?value={n}
  • Submit a metric value to a chart
  • Creates chart/namespace if needed
  • Records metric at current time

View Charts

GET /{ns}/{chart}
  • View interactive chart visualization
  • Shows all submitted data points

Example

charts.somnial.co/app/cpu

Browse Namespace

GET /{ns}
  • List all charts in namespace
  • Browse available metrics

Example

charts.somnial.co/app
